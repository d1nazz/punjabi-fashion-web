# Punjabi Fashion — project phases

Short roadmap for moving from a clean shell to a full South Asian ecommerce experience (inspired structurally by premium fashion sites, not copied).

1. **Cleanup foundation** — Remove template/tooling noise; reliable `dev` / `build`; documented active app.  
2. **Brand design system** — Tokens, typography, spacing, and components aligned to Punjabi Fashion.  
3. **Homepage rebuild** — Premium hero, featured collections, trust blocks.  
4. **Navigation and mega menu** — Women’s, men’s, kids, accessories, jewelry, sale, etc.  
5. **Collection pages** — Category and collection landings.  
6. **Product cards and product detail** — Grids, PDP, size/color where needed.  
7. **Cart, wishlist, search** — Client UX first; backend later.  
8. **Store info, contact, appointment** — Hours, location, forms.  
9. **SEO metadata and structured data** — Titles, descriptions, product schema when catalog is real.  
10. **Ecommerce integration** — Stripe Checkout via Netlify (see README): Canadian shipping-only, tiered Standard / Free shipping at $340 CAD subtotal.
